Noteworthy

The following technologies were deployed in the making of this project:

  • Web components were used to create custom elements. They were utilized for the website's header, footer, navigation, and pet card objects.
  • Bulma was used to provide consistent and pleasant styling throughout the website.
  • Both the Petfinder API and the Mapbox API were used to collect and display information about adoptable pets. Specicially, the Mapbox API allowed for pets to be displayed on a map based on a location entered by the user.
  • Fetch and promises were used to communicate with both APIs.
  • Font Awesome was used to obtain basic icons.
  • Google Fonts was used to obtain a unique font for the website.

My work on this project goes outside what was covered in class in several ways, including:

  • The use of two separate APIs to get information about adoptable pets and their locations.
  • Favoriting functionality that toggles whether a pet is in your favorites without having to explicitly clear it from the favorites page.
  • The ability to view results as either a list or as markers on a map.
  • The ability to initiate a search with the ENTER key.
  • Next and previous page buttons that disable if you are on the first or last page respectively.
  • A custom "No Image Found" image for API results which are missing a photo.
  • A custom font from Google Fonts
  • Pertinent warnings and indicators for the user if they are doing something incorrectly

Grading

As per the rubric, all requirements appear to have been met. For ways in which I went above and beyond, see the Noteworthy section above.


To Do

Given additional time, I would have liked to accomplish at least the following:

  • Additional controls, such as the option to search by age or gender.
  • Some sorting options for the user, such as the ability to sort by oldest or newest entries. Currently, the app defaults to showing the most recent pets to be put up for adoption first.
  • More sofisticated controls for searching for a breed. Ideally it would be a combination of a search bar and a dropdown. Given the number and complexity of breed types, it would afford the user more control over the results they receive if they are not familiar with animal breeds.

Proposal

API

The Petfinder API provides information for adoptable pets, including their location, breed, age, gender, and more.


About

This app will allow users to search for adoptable pets by:

  • Location
  • Species
  • Breed
  • Size
  • Gender
  • Age
  • etc.

Users will also be able to view a map of pets if they enter their location, as well as add pets to their favorites for later.
