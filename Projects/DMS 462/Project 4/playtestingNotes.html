      • Lose access to that resource when your ship moves off of the planet

      • Trading? - only to a player on an adjacent planet? Trading colonies?
  • Location of resources?
    • Should it be random? Fixed?
    • Some can be fixed, others random (stars for energy might need to be in fixed locations)

11/13/18
  • Attacking
    • Based on a players weapon system, they will need to roll between a range of values to win
      • Level 1: 2-3
      • Level 2: 2-4
      • Level 3: 2-5
  • Victory Point system
    • Attacking player wins: +1 victory point to attacker, -1 to opponent
    • Attacking player loses: -1 victory point to attacker, +1 to opponent
    • Each home planet a player has control of will give them 3 victory points
      • Cannot lose these unless you lose the home planet
  • Resources
    • Names and numbers to be determined still?
    • Each ship can hold 2 resources
    • Each home planet can hold 3 resources
